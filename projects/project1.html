Case Study

Beverage Outlier Dashboard

A dashboard built to identify unusual beverage sales movement across stores using normalized rate-of-sale logic.

SQL Tableau Stats Operations Analytics
View Code Live Demo

Overview

This project was designed to help operators quickly identify meaningful changes in beverage performance across stores. Instead of relying on raw sales alone, it used normalized rate-of-sale logic to make fairer comparisons.

Problem

Raw sales changes can be misleading because traffic, mix, and store volume vary widely. The goal was to surface true outliers, not just noisy differences.

Approach

I built a reporting layer that normalized performance, compared stores against expected behavior, and highlighted unusual movement for faster review.

Outcome

The result was a cleaner operational view that made it easier to investigate change, explain variance, and focus follow-up where it mattered most.
